# Twilight Treasures (Advanced Database System)

## Overview
Twilight Treasures is a database-driven management system for Twilight-themed books, memorabilia, and collectibles.
It supports managing products, staff, customers, and orders in a structured system.

## Modules
- Product Management
- Order Management
- Staff Management
- Customer Management

## Key Features
- CRUD operations (Insert, Update, Delete)
- DataGrid display + form-based editing
- Product catalog with image handling (Select Picture)
- Organized navigation with a main menu interface

## Tools / Technologies
- Microsoft Access (Database)
- Visual Studio (VB.NET / Windows Forms)
