# SiswaRide (HCI Project)

**Course:** TTTK2093 Human Computer Interaction  
**Project Type:** Mobile App Prototype (Paper + High-Fidelity)  
**Tool:** Figma  

## Overview
SiswaRide is a student-focused ride-sharing application designed for UKM students to solve campus transportation issues such as safety concerns, long waiting times, and high ride costs.

## Key Features (Prototype)
- Passenger & Driver mode
- Ride booking + scheduled pickup
- Points reward system
- Cashless payment (QR pay + card)
- Safety-focused concept (UKM students only)
- Ride history + ratings + support
