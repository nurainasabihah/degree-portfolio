# Diena Design (Database ERD Case Analysis)

**Course:** TTTT1964 Database  
**Project Type:** Case Analysis + ERD Design (Conceptual & Logical)  

## Overview
This project focuses on designing a database structure for **Diena Design**, an online Shopee store selling animal-themed button badges.
The goal is to model the business process and create a clear ERD for inventory, orders, invoices, and transactions.

## Deliverables
- Business rules definition
- ERD (Conceptual level)
- ERD (Logical level)
- Primary Key & Foreign Key identification
- Data dictionary

## Entities Identified
- Customer
- Order
- Product
- Invoice
- Transaction

## Key Business Rules (Summary)
- A customer can make one or many orders
- Each order belongs to one customer
- Each order has one invoice
- Each invoice has one transaction
- Orders consist of one or many products

## Report
- Diena Design Case Analysis Report (ERD)
