# Roomly (OOSE Project)

**Course:** TTKK2023 Object-Oriented Software Engineering  
**Project Type:** Simplified Airbnb Property Booking System  
**Architecture:** MVC (Model-View-Controller)

## Overview
Roomly is a streamlined Airbnb-like platform that allows **Guests** to browse and book properties, and **Hosts** to manage listings. The system focuses on core booking features with improved clarity in pricing, communication, and cancellation policies.

## Key Features
### Guest
- Register / Login / Logout
- Browse listings + view property details (including images)
- Book property + make payment
- View booking list + cancel booking
- Leave review & rating
- Messaging with host

### Host
- Register / Login / Logout
- Manage property (Add / Edit / Delete)
- Respond to messages
